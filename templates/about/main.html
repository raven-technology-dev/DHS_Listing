About this Website

Dimond HS (Dimond High School) Listings is a website where members of Dimond High School can post information about clubs, and other things. This website is created by students at Dimond High School, and Raven Technology, a local website development company. This project has no official affiliation with Dimond High School, or the Anchorage School District. For more information, please contact us.

This website was created because Dimond High School did not have a quality listing platform before this platform. Dimond High School has some web-pages with some information about relevant listings on them, but those pages tend to have little information, or other issues with them. That's why this website was created, so members of Dimond High School have a good way to see relevant things about the school.

This website is still very new, so more features will be added shortly. However, Raven Technology is planning on implementing some new features, sometime in the future. Here is a list of what we are working on:

  • Feature Request forms.
  • User Interface Design improvements.
  • Search filtering.
  • And other features as requested by our users.

If you have an idea for a feature for this website, please feel free to contact us. Any support for this project is appreciated.

This project has some staff that work to make this platform happen.
